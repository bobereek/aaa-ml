# Suggest: классификация категорий товаров по названиям объявлений товаров Авито

## Описание задачи

По названию объявления товара (`title`) предсказать категорию (`category_id`) для карточек маркетплейса. Помимо точного попадания оценивается попадание правильной категории в топ-3 рекомендаций.

## Что сделано

- Проведен EDA и подготовка датасета.
- Использованы несколько подходов к векторизации:
  - TF-IDF по символьным n-граммам (устойчив к опечаткам и морфологии).
  - Word2Vec (усреднение эмбеддингов слов).
  - FastText (эмбеддинги с учетом субслов).
  - Комбинации эмбеддингов с TF-IDF.
- Обучены модели `LogisticRegression` для мультиклассовой классификации с балансировкой классов.

## Метрики

- `accuracy` — доля точных попаданий в правильную категорию.
- `accuracy@3` — доля примеров, где правильная категория попала в топ-3 по вероятности.

## Результаты

Лучшее качество показала модель на TF-IDF признаках. Добавление Word2Vec/FastText и их комбинаций не дало устойчивого улучшения (скорее всего, из-за ограниченного объема данных для обучения эмбеддингов).

## Использованные инструменты

- Python, pandas, numpy
- NLTK, pymorphy3
- scikit-learn
- gensim (Word2Vec, FastText)
